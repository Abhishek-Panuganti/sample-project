Replace "index.html" with the appropriate path to your index.html file within your Git repository, and replace "your-bucket-name" with the name of your S3 bucket. This buildspec.yaml file specifies a single build phase (build), where it runs the command aws s3 cp to copy the index.html file to the specified S3 bucket.
